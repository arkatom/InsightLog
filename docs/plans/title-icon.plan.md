# タイトルアイコン追加実装計画

**想定所要時間:** 3-5分
**難易度:** 低
**前提条件:** なし

## 概要

ヘッダーのタイトル「InsightLog」の左横に、赤い円（#EF4444）のインラインSVGアイコンを追加する。

## 実装対象ファイル

1. **src/components/layout/Header.tsx** — タイトル横にインラインSVGを追加

## 実装手順

### 1. Header.tsxの修正

`<h1>` タグの中、テキスト「InsightLog」の直前にインラインSVGを追加する：

* SVGサイズ: `1em`（タイトル文字と同じ大きさに自動追従）
* 色: `#EF4444`（Tailwind CSSのred-500相当）
* 形状: シンプルな円（`<circle>`）

## 受け入れ基準

* `src/components/layout/Header.tsx` にインラインSVGが追加されている
* 画面上で、タイトル「InsightLog」の左横にタイトル文字と同じ大きさの赤丸が表示されている

## 検証方法

```bash
npm run dev
```

* [ ] ヘッダーのタイトル左横に赤丸が表示されているか確認

## 注意点

* インラインSVGを使用するため、外部ファイルの追加は不要
* `1em` 指定でフォントサイズに自動追従する
